# navio_blsct

## Requirements
- Python 3.12 or 3.13
- macOS (arm) or Linux

## Installation

```bash
pip install navio_blsct
```

## Usage

1. Import the library

```Python
import blsct
```

2. Refer to [a test script](https://github.com/nav-io/libblsct-bindings/blob/main/ffi/python/tests/test_all.py) for usage examples
