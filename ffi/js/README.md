## Preparation  

### Install `libblsct.a`, `libbls384_256.a` and `libmcl.a`

Follow steps described in the top-level [README.md](../../README.md)

### Install `node-gyp`

```bash
$ npm i -g node-gyp
```

## Building `blsct.node` module

```bash
$ make
```

## Running `test.js`
```bash
$ make run
```
